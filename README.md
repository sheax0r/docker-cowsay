# Docker Cowsay

Probably the most useless docker container ever invented.

## Installation
```bash
git clone https://github.com/sheax0r/docker-cowsay.git
cd docker-cowsay
docker build -t cowsay .
```
## Usage
Input:
```bash
docker run cowsay I LOVE COWS
```

Output:
```
 _____________
< I LOVE COWS >
 -------------
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
